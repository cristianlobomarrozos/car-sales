Stub class for backwards compatibility.

Since PHP 7 adds "float" as a reserved keyword, we can no longer have a class named that and retain PHP 7 compatibility. The original class has been renamed to "IsFloat", and this class is now an extension of it. It raises an E_USER_DEPRECATED to warn users to migrate.
